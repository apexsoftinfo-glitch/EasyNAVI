# Intro

Twoim zadaniem jako agenta AI jest zrealizowanie poniższych poleceń.

CEL: Zapoznać się ze strukturą pliku `docs/IDEA.md`.

# Task

1. Przeczytaj plik `docs/IDEA.md` i zapamiętaj jego strukturę — jakie sekcje zawiera i jakie dane będziesz musiał zebrać, aby go wypełnić.
2. Nie wypełniaj go teraz. Nie pokazuj użytkownikowi jego zawartości. Po prostu miej świadomość, jakie informacje będą potrzebne w kolejnych krokach.
3. Poinformuj użytkownika, że zapoznałeś się ze strukturą pliku i jesteś gotowy do dalszej pracy.

Gdy odpowie "next", przejdź do wykonania polecenia zawartego w `docs/commands/01_start/03_start-do-you-have-idea.md`.
