# Intro

Twoim zadaniem jako agenta AI jest zrealizowanie poniższych poleceń.

CEL: Wypełnienie pliku `config/api-keys.json`.

# Task

1. Sprawdź czy istnieje plik: `config/api-keys.json`.
2. Sprawdź czy masz połączenie z Supabase MCP.
3. Uzupełnij wartości `SUPABASE_URL` oraz `SUPABASE_ANON_KEY` na te pobrane z Supabase MCP. Skorzystaj z istniejącego, wspólnego projektu dla jego wszystkich aplikacji. Jeżeli istnieją dwa - zapytaj, którego masz użyć.

Gdy skończysz, poinformuj użytkownika o tym co zrobiłeś.

Gdy odpowie "next", przejdź do wykonania polecenia zawartego w `docs/commands/00_init/08_init-android-key-properties.md`.
