# ZADANIE: 11_design_stylize_other_screens

## CEL
Dopasuj wizualnie istniejące ekrany do ustalonego stylu opisanego w `docs/DESIGN.md`.

## KROKI DO WYKONANIA:
1. Znajdź inne ekrany w aplikacji, takie jak np.:
   - Welcome Screen — `lib/features/welcome/ui/welcome_screen.dart`
   - Profile Screen — `lib/features/profiles/presentation/ui/profile_screen.dart`
   - Logowanie — `lib/features/auth/presentation/ui/login_screen.dart`
   - Rejestracja — `lib/features/auth/presentation/ui/register_screen.dart`
   - Usuwanie konta — `lib/app/profile/presentation/ui/delete_account_confirmation_screen.dart`
   i inne dostępne dla użytkownika. Ekranów developerskich nie ruszaj.
2. Dopasuj je wizualnie do stylu z `docs/DESIGN.md`.
3. Uruchom `flutter analyze`. Jeśli są błędy — napraw je.
4. Wykonaj commit.
5. Poinformuj użytkownika o zmianach i o ukończeniu etapu `design`. Zaproponuj przejście do kolejnego etapu `plan` i poleć mu napisanie `next`.

## FINISH
Gdy użytkownik odpowie `next`, przejdź do `docs/commands/04_plan.md`.
