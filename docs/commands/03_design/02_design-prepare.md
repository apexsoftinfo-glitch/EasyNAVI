# ZADANIE: 02_design_prepare

## CEL
Rozpisz szczegółowe plany implementacji dla każdego z 3 zaplanowanych wariantów wizualnych.

## KROKI DO WYKONANIA:
1. Dla każdego wariantu zapisz szczegółowy plan implementacji do odpowiedniego pliku, nadpisując sekcję `## PLAN IMPLEMENTACJI`:
   - Wariant D → `docs/commands/03_design/03_design-build-d.md`
   - Wariant E → `docs/commands/03_design/04_design-build-e.md`
   - Wariant F → `docs/commands/03_design/05_design-build-f.md`
   Nie zmieniaj sekcji `## CEL`, `## KROKI DO WYKONANIA` ani `## FINISH`.
2. Wykonaj commit.
3. Poinformuj użytkownika, że plany są gotowe. Poleć mu napisanie `next`.

## FINISH
Gdy otrzymasz `next`, przejdź do `docs/commands/03_design/03_design-build-d.md`.
