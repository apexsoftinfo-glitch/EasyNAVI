# ZADANIE: 01_test_setup

## CEL
Przygotowanie środowiska do testów integracyjnych.

## KROKI DO WYKONANIA:
1. Sprawdź czy nie ma niezacommitowanych zmian — jeśli są, uruchom `flutter analyze`, napraw ewentualne problemy i zacommituj.
2. Powiedz użytkownikowi, żeby odpalił emulator lub symulator (np. iOS Simulator lub Android Emulator). Testy integracyjne wymagają uruchomionego urządzenia.
3. ZATRZYMAJ SIĘ i czekaj aż użytkownik potwierdzi, że emulator działa.

## FINISH
Gdy użytkownik potwierdzi, przejdź do `docs/commands/06_test/02_test-integration.md`.
