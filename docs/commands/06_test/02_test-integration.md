# ZADANIE: 02_test_integration

## CEL
Uruchomienie testów integracyjnych na emulatorze.

## KROKI DO WYKONANIA:
1. Uruchom `flutter test integration_test/` na podłączonym emulatorze/symulatorze.
2. Jeśli jakieś testy nie przechodzą — napraw kod, uruchom `flutter analyze`, ponów testy.
3. Powtarzaj aż wszystkie testy przejdą.
4. Jeśli były poprawki — zacommituj.
5. Poinformuj użytkownika o wynikach. Poleć mu napisanie `next`.

## FINISH
Gdy napisze `next`, przejdź do `docs/commands/06_test/03_test-manual.md`.
