# ZADANIE: 05_publish_save_icon

## CEL
Użytkownik generuje grafiki AI i zapisuje ikonę w projekcie.

## KROKI DO WYKONANIA:

1. Powiedz użytkownikowi, żeby teraz wygenerował grafiki przy użyciu promptów z dołu pliku `docs/PUBLISH.md`. Niech użyje generatora obrazów (np. Nano Banana, Midjourney, DALL-E) i załączy screenshot swojej aplikacji jako inspirację wizualną.
2. Gdy wybierze najlepszą ikonę, ma ją zapisać w projekcie w lokalizacji `assets/images/icon/icon.png` (dokładnie taka nazwa i rozszerzenie).
3. ZATRZYMAJ SIĘ i czekaj aż użytkownik potwierdzi, że zapisał ikonę.

## FINISH
Gdy potwierdzi, poleć mu napisanie `next`.

Jak napisze `next`, dopiero wtedy zapoznaj się z plikiem `docs/commands/07_publish/06_publish-icon-gen.md` — nie wcześniej!
